# Desain-Kelas-Hacker-React-
Website X-Code Training (React + Tailwind CSS) 🚀 Ini adalah repositori untuk proyek landing page X-Code Training, sebuah platform kursus di bidang Cyber Security. Proyek ini merupakan hasil konversi dari halaman HTML &amp; CSS statis menjadi aplikasi web modern yang dinamis dan modular menggunakan React.js, Vite, dan Tailwind CSS.
